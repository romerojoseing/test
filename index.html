Sobre Mí.

Si te preguntas quien soy ...
Soy Romero, desarrollador web especializado en front-end autodidacta de Venezuela.
Durante mi etapa como estudiante universitario trabajé como "desarrollador", durante un poco más de 2 años donde desarrollaba sitios web con temas personalizados de Wordpress, HTML y CSS ... Sentía que no estaba aprendiendo o mejorando, me sentí estancado.
En febrero del 2019 decidí que era hora de un cambio, y mejorar mis habilidades en el desarrollo front-end. Luego de obtener mi título universitario como Ingeniero de Sistemas, decidí dar el siguiente paso y tomar cursos especializados para mejorar mis conocimientos: Master en CSS: Responsive, SASS, Flexbox, Grid y Bootstrap 4, Master en JavaScript: Aprender JS, jQuery, Angular 9, NodeJS, Master en PHP, SQL, POO, MVC, Laravel, Symfony, WordPress.
La forma de mejorar cualquier habilidad es Practicando, así comencé a crear proyectos personales para tener un portafolio , junto a esto siempre estoy creando proyectos que sirvan para compartir conocimiento, estos proyectos los comparto en repositorios de github.
Mi stack o tecnologías más usadas:
HTML5 - CSS3 - JAVASCRIPT - PHP - SASS - BOOTSTRAP 4 - CSS GRID - FLEXBOX - NODEJS - TYPESCRIPT - ANGULAR - LARAVEL - SQL - WORDPRESS
